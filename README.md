# House Control Telegram Bot (Raspberry Pi 3 + Node.js)
with user-friendly replyMarkup management

* Live-version: **@HouseController_bot**

## Functions:
  * **Controlling Raspberry Pi modules** 
  * **Getting data from sensors**
  * **Admin-only access** 
  * **Admins list**
  * **Activity log**
  * **Changing admin password**  
  * **Alarm in case fire or gas detection**
